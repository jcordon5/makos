# Confidence Scale

- `0.00 - 0.39`: hipótesis inicial, no consolidar.
- `0.40 - 0.69`: hallazgo plausible, mantener en inbox/workspace.
- `0.70 - 0.84`: listo para revisión de consolidación.
- `0.85 - 1.00`: conocimiento estable y reutilizable.

## Regla operativa

Para promover a `04-knowledge`, objetivo mínimo: `confidence >= 0.70` y evidencia enlazada.
