# Glossary

- Procedure: secuencia reutilizable para una tarea repetitiva.
- Skill: instrucción operativa portable para usar herramientas o aplicar reglas.
- Knowledge Note: conocimiento curado y navegable.
- Memory Entry: observación contextual de humanos/agentes.
- Scratchpad: espacio temporal de trabajo activo.
- Review Item: ítem explícito para revisión humana.
- History Entry: trazabilidad de acciones, decisiones o cambios.
