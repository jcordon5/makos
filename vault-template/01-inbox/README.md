# Inbox

`01-inbox` es la zona de aterrizaje para:

- contenido de baja confianza
- propuestas sin validar
- material pendiente de promoción o descarte

## Flujo

1. Capturar en inbox.
2. Asignar `review_due` y `owners`.
3. Revisar en `10-human-views/review-queue.md`.
4. Promover a `04-knowledge` o archivar.
